## 简单实践步骤
1. 看文档怎么使用 
>  [heml文档](https://heml.io/docs/getting-started/usage)

2. npm install
> ``` npm install heml -g ```

3. 编写了一个实验文档 [**email.heml** ](email.heml)

4. 使用命令

    + ``` heml develop email.heml --open```
  
    用于将你编写你的文件渲染，在浏览器中可以看到

    + ```heml build email.heml```

    用于将你编写好的email.heml 形成对应的 email.html 文件。同一个目录下产生

    + ```heml --help```

    CLI 帮助命令 ，目前貌似就这几个命令
